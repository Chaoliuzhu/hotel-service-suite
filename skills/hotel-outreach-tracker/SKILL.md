# hotel-outreach-tracker · 销售外拓每日追踪

## 定位

替代现有"管人型"外拓台账，升级为"AI赋能型"外拓系统。核心理念：不是管你有没有去拜访，而是AI帮你算好今天最该去拜访谁、怎么谈、谈完怎么跟。

## 核心算法

### 客户评分模型

```
总分 = 基础分 × 信号加权 × 紧迫度系数

基础分(0-100):
  - 企业规模(营收): >5亿=40, 1-5亿=30, 0.5-1亿=20, <0.5亿=10
  - 行业会议需求: 金融/政府/医药=30, IT/物流=25, 制造=20, 其他=15
  - 历史合作: 已有协议=20, 有过接触=10, 全新=5
  - 内线存在: 百达屋会员在该企业≥2人=10, 1人=5, 无=0

信号加权(×1.0-2.0):
  - 新注册/扩租: ×1.5
  - 融资事件: ×1.3
  - 人事变动(新任领导): ×1.2
  - 无新信号: ×1.0

紧迫度系数(×0.8-1.5):
  - 竞品合约即将到期(3个月内): ×1.5
  - 年度会议规划期(Q4): ×1.3
  - 正常: ×1.0
  - 淡季: ×0.8
```

### 商机漏斗状态机

```
新线索 → 初步接触 → 需求确认 → 方案提交 → 商务谈判 → 签约 → 续约
    ↓          ↓          ↓          ↓          ↓
  冷-培育   温-跟进    温-跟进    热-推进    热-推进
                    ↓
               沉睡-唤醒(>14天无跟进)
```
